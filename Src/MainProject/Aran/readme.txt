KNOWN ISSUE
========================================================================================================================

ModelExporter (ARN format exporter for 3ds Max 9/2008)

	UPDATED December 30th, 2007
	----------------------------------------------------
	- (Physique modifier) This is not an accpetable modifier to be converted to skinning data;
	Implemented algorithm in the ARAN engine assumes that there are fixed correspondance between vertices
	and effective bones. However, Physique modifier does not seem to have these requirements.
	Use 'Skin' modifier instead.
	




	- (Geometry; guess) Geometry generated by 'Mirror' feature do not have the proper vertex normal value;
	Diffuse color is darker than its mirror source geometry. Use 'Symmetry' modifier instead.
	
	- (Physique modifier) 'Deformable' blending method works differ from 3ds Max 9 and game engine;
	Use 'Rigid' blending method instead ... SEE LATEST INFO
	
	- (Physique modifier) Target model(geometry) should have single material information;
	that is, single texture(UV map) Or the skeleton animation should not work properly ... SEE LATEST INFO

	- (Axis) Coordinate system is different between 3ds Max 9 and DirectX; Right exact conversion needed
