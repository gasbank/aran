DirectX SDK (November 2007) 필수 파일
===================================================================

빌드 및 실행에 필수적인 파일만 모아 놓은 것

	x64 관련 제외
	다큐먼트 제외
	샘플 제외


Developer Runtime이란 것이 있는데, 아마도 DX SDK가 설치되지 않은

컴퓨터에서는 따로 필요할 수도 있을 것 같아서 포함시켜 두었다.

일반적으로 DX 게임 개발용 컴퓨터에는 DX SDK가 있을 것이라는 가정을

하지만, 그렇지 않은 컴퓨터에서는 Developer Runtime에 있는 DLL을

Working 폴더에 복사해서 실행해보길 바란다. 참고로, Developer Runtime은

Debug 빌드인 경우에만 사용되는 DLL이다.



Runtime이란 폴더는 내가 직접 만든 것인데, Release 빌드인 경우에는

이 DLL 파일이 Working 폴더에 있어야 할 것이다.
