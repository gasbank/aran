Working 폴더에서 실행시키기 위한 Aran 의존성
==================================================================

- Visual C++ Redistributable SP1

	Visual Studio 2005 (SP1)으로 작성된 C++ 응용 프로그램은 이게 있어야 실행이 된다.
	다만 실행하려는 컴퓨터에 VS2005 SP1이 이미 설치되어있다면 굳이 필요한 것은 아니다.

- DirectX 9.0c

	이것은 대부분 Windows XP SP2가 설치된 컴퓨터라면 설치되어있다고 봐도 무방하다.

- d3dx9 관련 DLL

	DX SDK 버젼이 올라갈때마다 바뀌긴 하지만 일반적으로 d3dx9_36.dll 같은 형태이다.
